La Pensiunea Select din Alba Iulia găsiți echilibrul perfect între calitate și serviciile de cazare. Pensiunea este clasificată la 3 stele și este amplasată pe strada Biruinței Nr. 72.

Pensiunea Select Alba Iulia

Pensiunea Select este amplasată la doar 2 km de centrul orașului și la 1,5 km de frumoasa cetate de tip Vauban, Alba Carolina. Construcția este finalizată în 2014 și pune la dispoziția dvs. 22 camere. Pe lângă ospitalitate și confort, aveți parte și de acces gratuit la internet Wi-Fi, parcare privată gratuită și mic dejun inclus în prețul camerei.

Cazare și servicii

Pensiunea Select oferă spre cazare 18 camere și 2 apartamente, fiecare având baia proprie, aer condiționat, TV, Wi-Fi gratuit, micul dejun este inclus în prețul fiecărei camere. Condițiile foarte bune de cazare sunt completate de un restaurant și-o sală spațioasă pentru evenimente.

Unitatea de cazare este structurată în:

  • Camere Single
  • Camere Twin
  • Camere Duble
  • Camere Triple
